Request Header Parser Microservice

Microservice ini mengekstrak informasi dari header permintaan HTTP dan mengembalikannya sebagai objek JSON.

Contoh Penggunaan:

[URL Proyek]/api/whoami

Contoh Output:

{
  "ipaddress": "192.168.1.1",
  "language": "id-ID,id;q=0.9,en-US;q=0.8,en;q=0.7",
  "software": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/96.0.4664.110 Safari/537.36"
}

Proyek ini dibuat untuk FreeCodeCamp Back End Development and APIs Certification
